📌 Project Title
Selenium Cucumber BDD Automation Framework

A scalable and maintainable Test Automation Framework built using Selenium WebDriver, Java, Cucumber BDD, TestNG, Maven, and Page Object Model (POM).

This framework automates end-to-end user workflows and follows industry-standard automation design practices to ensure reusability, maintainability, and scalability.
